telegram-bot-for-college

🚀 telegram-bot-for-college

Python · Aiogram · Redis · Docker Compose

Telegram-бот для колледжа: сам определяет текущий тип недели (числитель/знаменатель) и присылает расписание перед парой. Отвечает на /para (что сейчас за пара), показывает все пары на день командами all/next, отменяет отдельные пары на сегодня через /cancel и пингует подписавшихся перед началом занятия (/pingme, /pingwho). Состояние — в Redis, деплой — через docker-compose.
