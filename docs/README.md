# Ку пивас

Файловая структура:
 - [`code/`](/code) - папка с кодом.
   - [`Bot.py`](/code/Bot.py) - бот.
   - [`System.py`](/code/System.py) - Вся логика здесь.
   - [`DataBase.py`](/code/DataBase.py) - Все функции по считыванию файлов здесь.
   - [`ping.txt`](/code/ping.txt) - Список тех кого пинговать.
   - [`Url.txt`](/code/Url.txt) - Ссылки на уроки.
   - [`Schedule.txt`](/code/Schedule.txt) - Расписание пар.
 - [`docs/`](/docs) - Тут все файлы для гит хаба.
 - [`main.py`](/main.py) - Запуск бота.
 - [`requirements.txt`](/requirements.txt) - библиотеки которые надо скачать.
